🌐 Aswin M Kumar - Personal Portfolio Website
Welcome to the source code of my personal portfolio website! This site serves as a showcase of my skills, projects, achievements, and passion for Electrical and Electronics Engineering, embedded systems, AI, and app development.

🔗 Live Preview: aswin-m-kumar.github.io

🧠 About This Website
This is a fully responsive, animated, and interactive one-page portfolio built using HTML, CSS, and JavaScript. It is enhanced with the power of the Google Gemini API to create a more engaging user experience. The design is modern, clean, and built from scratch with no external CSS frameworks.

✨ Features
⚡ Animated Hero Section: A dynamic hero section with a typing effect to introduce myself.

resume Resume/CV Link: A prominent link to my resume so visitors can easily learn more about my qualifications.

🧾 Comprehensive Sections: Includes detailed sections for About, Skills, Featured Projects, and a gallery of my Certificates & Achievements.

📱 Fully Responsive: The layout is optimized for all devices, from mobile phones to desktop computers.

🤖 Gemini API Integration:

Smart Contact Form: The contact form uses the Gemini API to generate an intelligent, contextual auto-reply to user messages, providing instant feedback.

🎨 Modern UI/UX: Features smooth scrolling, parallax effects, and hover animations for an engaging user experience.

🌗 Theme Toggle: A dark/light mode theme toggle to suit user preference.

🤖 Gemini API Integration
This portfolio uses the Google Gemini API to power the Smart Contact Form.

When a visitor sends a message, the content is sent to the Gemini API.

The API generates a brief, friendly, and context-aware response based on the message content.

This provides a more personal and interactive experience than a standard "message sent" confirmation.
