🌐 Aswin M Kumar - Personal Portfolio Website
Welcome to the source code of my personal portfolio website! This site serves as an interactive showcase of my skills, projects, achievements, and passion for Electrical and Electronics Engineering, embedded systems, AI, and app development.

🔗 Live Preview: aswin-m-kumar.github.io

🧠 About This Website
This is a fully responsive, animated, and interactive multi-page portfolio built using HTML, CSS, and JavaScript. It is enhanced with a "Fun Zone" with custom-built games. The design is modern, clean, and built from scratch with no external CSS frameworks.

✨ Features
Multi-Page Architecture: The site is organized into logical sections for a clean user experience:

A main landing page for a comprehensive overview.

A dedicated Experience Page for detailed internship and program information.

A Fun Zone with interactive games.

🎮 Fun Zone: A unique section to showcase my development skills in a creative way, featuring:

Circuit Master: A logic puzzle game directly related to my Electrical Engineering background.

Orbital Dodge: A fast-paced HTML canvas game to test reflexes.

📱 Fully Responsive Design: The layout is optimized for a seamless experience on all devices, from mobile phones to desktop computers.

🧾 Comprehensive Sections: Includes detailed sections for About, Skills, Featured Projects, and a gallery of my Certificates & Achievements.

🎨 Modern UI/UX: Features smooth scrolling, parallax effects, and hover animations for an engaging user experience.

🌗 Theme Toggle: A dark/light mode theme toggle to suit user preference.

🔧 How to Use
Download or Clone the Repo:

git clone https://github.com/aswin-m-kumar/aswin-m-kumar.github.io.git

Navigate to the Directory:

cd aswin-m-kumar.github.io

Open index.html:
Open the index.html file in your web browser to view the portfolio locally.
