## 🧠 About This Website

This is a **fully responsive, animated, and interactive** multi-page portfolio built using HTML, CSS, and JavaScript. It features a unique **"Fun Zone"** with custom-built games. The design is modern, clean, and built strictly from scratch without relying on any external CSS frameworks like Bootstrap or Tailwind.

---

## ✨ Features

- **📑 Multi-Page Architecture**: The site is logically organized to provide a clean user experience:
  - **Main Landing Page**: A comprehensive overview of my journey and skills.
  - **Experience Page**: Detailed insights into my internships and programs.
  - **Fun Zone**: A dedicated area for interactive mini-games.

- **🎮 Fun Zone**: A unique creative space showcasing my development logic, featuring:
  - **Circuit Master**: A logic puzzle game testing Electrical Engineering fundamentals.
  - **Orbital Dodge**: A fast-paced HTML canvas game to test your reflexes.

- **📱 Fully Responsive Design**: Seamlessly adapts to all screen sizes, from mobile devices to ultrawide desktop monitors.

- **🧾 Comprehensive Sections**: Detailed areas covering About Me, Skills, Featured Projects, and a dedicated gallery for Certificates & Achievements.

- **🎨 Modern UI/UX**: Includes elegant smooth scrolling, parallax effects, and interactive hover animations for an engaging experience.

- **🌗 Theme Toggle**: Built-in dark/light mode toggle adapting to user preferences.

---

## 🚀 What's New in This Generation

Compared to the previous version, this portfolio has been completely reimagined to offer a richer, more engaging experience:

- **New "Fun Zone"**: Added a dedicated games section featuring *Circuit Master* and *Orbital Dodge* built from scratch.
- **Enhanced Multi-Page Layout**: Transitioned to a clean, multi-page architecture (Main Page, Experience, and Games) for better content organization.
- **Overhauled UI/UX & Responsiveness**: Completely updated the CSS to support advanced smooth scrolling, parallax effects, and strict mobile responsiveness without any external frameworks.
- **Dark/Light Mode Toggle**: Built a custom, system-aware theme switcher for personalized viewing.

## 🔧 How to Run Locally

If you wish to explore the code or run this portfolio on your local machine, follow these simple steps:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/aswin-m-kumar/aswin-m-kumar.github.io.git
   ```

2. **Navigate to the Directory**
   ```bash
   cd aswin-m-kumar.github.io
   ```

3. **Open the site**
   Open the `index.html` file in your preferred web browser, or use an extension like **Live Server** in VS Code for a better development experience.
